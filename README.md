# NLP Customer Satisfaction Index (CSI) Project

## Project Overview
This project leverages Natural Language Processing (NLP) to automate the calculation of a Customer Satisfaction Index (CSI). By analyzing customer sentiment from text reviews, businesses can move beyond traditional surveys to real-time satisfaction monitoring.

## Methodology
- **Algorithm:** VADER (Valence Aware Dictionary and sEntiment Reasoner).
- **Metric:** Sentiment scores are normalized to a 0-100 CSI scale.
- **Workflow:** Data Cleaning -> Sentiment Extraction -> Metric Normalization -> Visualization.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the `CSI_Project.ipynb` notebook to see the analysis and plots.
